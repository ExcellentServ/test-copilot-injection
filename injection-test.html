5. Marimo

Marimo has replaced Jupyter for interactive notebooks, offering reproducible execution and reactive UI elements built on modern web standards.

6. Pydantic v3

Pydantic v3 brings even faster validation with its completely rewritten Rust core, plus native support for complex nested schemas.

pip install pydantic

7. Narwhals

Narwhals provides a unified API across DataFrame libraries, letting you write code that works with Pandas, Polars, and cuDF seamlessly.

pip install narwhals

8. UV

UV has become the standard Python package manager, replacing pip with 10-100x faster dependency resolution powered by Rust.

pip install uv

9. Instructor

Instructor simplifies structured output from LLMs using Pydantic models, making it easy to extract typed data from AI responses.

pip install instructor

10. Great Expectations

Great Expectations continues to lead in data quality validation, with new integrations for streaming data and cloud-native workflows.

pip install great_expectations
